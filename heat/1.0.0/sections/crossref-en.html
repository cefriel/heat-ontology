A Context is the specific environment, application domain, or scenario in which a tool or a construct is applied, providing the situational framework that influences how concepts are interpreted, measured, and utilized.

General Constructc back to ToC or Class ToC

IRI: https://w3id.org/heat#GeneralConstruct

A General Construct is a generalised concept that serves as a reference point for aggregating all related constructs from different tools, enabling the systematic organisation and comparison of equivalent constructs in different contexts.

Reference Theoryc back to ToC or Class ToC

IRI: https://w3id.org/heat#ReferenceTheory

A Reference Theory is a standardized, conceptual theory that aggregates related tools, models, or variants under a unified theoretical structure. It serves as the canonical reference point to which all specific implementations or adaptations (tools/models) are linked.
is in range of
refers to theory op

Toolc back to ToC or Class ToC

IRI: https://w3id.org/heat#Tool

A Tool is a structured conceptual system that explains phenomena by specifying key constructs and the relationships between them; each tool is defined by a set of constructs, which are defined through a series of measurement elements (questions) that allow for an objective evaluation of the phenomenon analysed by the tool.
is in domain of
has construct op, has context op, has model diagram dp, is original dp, refers to theory op
